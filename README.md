## Authors

## Run Locally

Clone the project

```bash
  git clone https://github.com/moonraker22/moon-and-stars
```

Go to the project directory

```bash
  cd moon-and-stars
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
